# 🛒 Análise de Cesta de Compras (Market Basket Analysis)

Projeto de análise de dados utilizando Python, SQL e Google BigQuery para identificar padrões de compra e associações entre produtos a partir de dados transacionais.

## 🎯 Objetivo
Identificar padrões de consumo e relações entre produtos utilizando técnicas de Market Basket Analysis, apoiando estratégias de vendas e recomendação.

## 🛠 Tecnologias utilizadas
- Python
- SQL
- Google BigQuery
- Pandas
- Mlxtend

## 📊 Análises realizadas
- Extração de dados do BigQuery
- Transformação e limpeza de dados
- Aplicação de regras de associação
- Identificação de padrões de compra

- Support
- Confidence
- Lift
